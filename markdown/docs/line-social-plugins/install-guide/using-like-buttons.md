# 「いいね」ボタンを設置する

## 概要

「いいね」ボタンは、「友だち追加」ボタンと一緒に設置することもできます。「友だち追加」ボタンと一緒に設置するには、LINE公式アカウントのIDが必要になります。

> [!WARNING]
> 注意
> AppleのITPポリシーにより、Safariで「いいね」ボタンが正常に動作しない場合があります。
